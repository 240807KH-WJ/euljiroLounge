당부사항

샘플 프로젝트를 전달받았다고 해서, "이거 그대로 포트폴리오로 써야겠다!"라고 생각하지 않았으면 합니다.
이 프로젝트를 공유드린 이유는, 프로젝트 구축과 설계 과정에서 어려움을 느끼는 분께 방향을 제시하기 위해서입니다.
이 프로젝트를 단순히 사용하는 데 그치지 말고,
분석하고, 스스로 수정해보며, 추가하고 싶은 기능이나 코드를 작성해 보면서 학습해 주시길 바랍니다.
지금은 열정을 가지고 적극적으로 학습해야야 할 가장 중요한 시기입니다.
현재의 상태에 안주한다면 취업뿐만 아니라, 취업 후의 성장에도 어려움을 겪을 수 있습니다.
본인이 미래에 어떤 모습으로 성장하고 있을지 한번 깊이 생각해 보셨으면 합니다.
응원하겠습니다.

현재 구현된 기능

  1. 로그인
  2. 회원가입
  3. LoginFilter
  4. 게시판 종류 Interceptor
  5. 각 게시판(심리상담/마인드포스팃/문의게시판) 목록 조회
  6. 마인드포스팃
    • 상세조회
    • 작성
    • 삭제
    • 좋아요 check/uncheck
  7. 심리상담
    • 상세조회
    • 작성
  8. 문의게시판
    • 상세조회
    • 작성 + 이미지

스스로 구현해야만 하는 기능

  2. 심리상담

    • 댓글
  3. 문의게시판

    • 삭제
